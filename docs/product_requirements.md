# RoomLight — Product Requirements & Prototype Scope

---

## 1. Product Requirements

| ID | Requirement Description | Type | Priority |
|----|------------------------|------|----------|
| REQ-001 | Hotel staff can set lighting settings for a room | Functional | Must |
| REQ-002 | Lighting settings can be applied to multiple rooms at once | Functional | Must |
| REQ-003 | Lighting settings can be updated centrally without visiting rooms | Functional | Must |
| REQ-004 | Guests can adjust room lighting easily | Functional | Must |
| REQ-005 | System provides basic lighting modes (e.g. Relax, Work, Sleep) | Functional | Should |
| REQ-006 | System reduces energy usage when a room is not in use | Functional | Must |
| REQ-007 | Staff can see the current lighting status of rooms | Functional | Should |
| REQ-008 | System works for different room types (e.g. standard, suite) | Functional | Should |
| REQ-009 | System is easy to use for non-technical staff | Non-functional (Usability) | Must |
| REQ-010 | System still works at a basic level if the network connection is lost | Non-functional (Reliability) | Must |

---

## 2. Prototype Scope

### 2.1 Prototype Description

The prototype shows how hotel staff can set lighting once and apply it to multiple rooms.

It is meant for hotel staff who want to manage lighting without going to each room.

The prototype focuses on central control and shows how changes are applied to multiple rooms at the same time.

The goal is to prove that RoomLight can make lighting management faster and easier.

---

### 2.2 What the Prototype Demonstrates

The prototype demonstrates these requirements:

- REQ-001: Staff can set lighting settings  
- REQ-002: Settings can be applied to multiple rooms  
- REQ-003: Central updates without visiting rooms  
- REQ-007: Staff can see room lighting status  

---

### 2.3 Why These Requirements

These requirements show the main idea of RoomLight:

**"Set once, apply everywhere."**

If this works in the prototype, it shows that the system can reduce manual work and improve how hotels manage lighting.
